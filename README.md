# woki

A blazingly fast, easy-to-use C++ library for GUI, 3D graphics, and compute.

Built on **WebGPU** for truly cross-platform rendering — desktop, web, and maybe MCUs someday.

## Build
```bash
make          # build
make test     # run tests
make run      # run studio
```
